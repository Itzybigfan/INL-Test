Github 사용하는 방법(간단) 

Step 1) Creating a repository

우측 상단에 "New repository"를 누르면 저장소 생성가능합니다.
public은 무료이고, private도 어느 정도는 무료입니다.
다만, 일정량 이상이라면 유료결제가 필요합니다. 
저장소를 만듭니다.

Step 2) Create a Branch
만약 같은 내용을 가진 파일에서, 새로운 버전을 만들고 싶다면 사용합니다.
Branch라고 쓰여있는 곳을 누르면, Find or Create branch가 나옵니다.
해당 칸에 새로 Branch 이름을 누르면 기존 파일과 같은 내용을 가진 Branch를 만들 수 있습니다.
Readme-edit이라는 새로운 브런치를 만들었습니다.
필요한 부분을 수정한 후, 원한다면 Master와 합칠 수 있습니다.

Step 3) Make and commit changes
커밋은 "수정"을 의미합니다. 
파일 우측에 있는 연필 모양 아이콘을 누르면 파일을 수정할 수 있습니다.
해당 파일을 원하는 내용을 수정하면, 해당 파일에 표시가 됩니다.


Step 4) Open a pull request
Pull request는 두 Branches의 차이점을 확인할 수 있습니다.
초록색은 추가된 내용, 빨간색은 사라진 내용입니다.
Pull request를 누른 뒤, New pull request를 누릅니다.
readme-edits와 master를 비교하기 위해 선택합니다.
변경될 내용을 저장하고 싶다면, create pull request를 누릅니다.
그 다음 코멘트를 달고 branch에 저장하면 됩니다.

Step 5) Merge your Pull request
Branch에서 변경한 내용을 Master에 반영하기 위한 작업입니다.
Merge pull request를 누르고, master를 누릅니다.
confirm merge를 누르면 master에 반영이 됩니다.
delete branch를 누르면 해당 branch는 삭제됩니다.
